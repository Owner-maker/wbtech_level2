Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Вывод: [3 2 3]

Дело в том, что при передаче слайса строк в функцию, копируется целиком значение структуры слайса
вместе с указателем на базовый массив. При инициалиации слайса s его длина равна 3, емкость также равна 3.

В функции modifySlice по нулевму индексу будет заданое значение "3", но после того, как была вызвана 
функция append и добавлено новое значение "4", был выход емкости за пределы длины и создан новый слайс строк
вместе с новым базовым массивом, и, несмотря на то, что результат был присвоен обратно в переменную i,
последующие изменения никак не отразятся на результирующем слайсе, к-й был передан в качестве аргумента,
т.к. переменная i -> хранится на стэке, и не ведет к слайсу строк, к-ый был передан

Чтобы изменения отразились на переданной в качестве аргумента слайсе, нужно как вариант возвращать этот слайс
из функции и присваивать в переменную в функции main.
```
