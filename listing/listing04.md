Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Вывод будет от 0 до 9 включительно с шагом 1 (каждое значение на новой строке)
Но после крайнего значение - дэдлок.

Дело в том, что основной поток main в бесконечном цикле читает значения с канала и
после того как пишушая горутина закончит свою запись (9 - последний элемент), основной поток
main заблокируется, ожидая значение с канала в резальтате чего Go понимает, что
горутина не остановится -> получаем дэдлок.

Выход -> после того, как пишущая горутина закончит писать в канал, закрыть поток, в рез-е
чего основной поток main перестанет читать канал и программа успешно завершиться с кодом 0.
(Под капотом в записи for n := range ch лежит проверка на ok - канал закрыт или нет)

```
