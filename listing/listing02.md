Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Вывод:
2
1

Порядок вызовов defer определяется их положением в стэке, т.е. если в определенной 
функции были объявлены несколько defer, первым выполнится тот, к-ый был объявлен последним (LIFO).

Результат вызовы функции test() = 2, т.к. в данном случае произошло замыкание отложенной функции с 
переменной x (возврщаемым значением), поэтому результат сохранится именно в ту область стэка, где и хрнилось возвращаемое значение.

В случае же вызовы функции anotherTest() замыкание также произойдет на переменной x, которая объявлена через var, 
но ее значение будет хранится в другой области стэка (не в той же, где хранится значение переменной x внутри отложенной функции),
поэтому даже после того, как отложенная функция отработает, значение останется прежним = 1.
```
